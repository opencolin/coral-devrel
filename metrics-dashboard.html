Metric families

Four buckets, twelve metrics.

Acquisition

How people find Coral

  • GitHub stars (delta/week)
  • Weekly installs (brew + sh + Windows)
  • Discord new members
  • Newsletter subscribers
Activation

From install to value

  • % configuring ≥1 source @ 7 days
  • % configuring ≥3 sources @ 30 days (north star input)
  • MCP registrations (Claude Code / Cursor / Cody / Continue)
Engagement

Active usage

  • WAD — queries executed/week
  • Source specs contributed/week
  • PRs merged from community
Commercial

Post-Q3 2026

  • Cohort → Cloud conversion rate
  • Coral Cloud ARR
  • Logo count
Instrumentation

What we ship in week 1 to make the dashboard real.

  1. Anonymous telemetry opt-in (default-on, easy opt-out) — captures install, sources configured, queries/week. Privacy: aggregated only, no SQL bodies, no row data.
  2. GitHub API harvester for stars / forks / PRs / contributors
  3. Discord analytics — members, daily active, message volume per channel
  4. Plausible / PostHog for docs + landing analytics
  5. Loops.so / Resend for newsletter analytics
  6. Quarterly metrics readout — public blog post + private investor update
